## Project Overview

This project is a simple machine learning deployment demo created while learning how to deploy ML applications.
It demonstrates the difference between Linear Regression and Polynomial Regression by visualizing how each model fits the same dataset.

The main goal of this project is learning deployment workflows, not building a production-level system.
